# DD 的个人网站

初二全栈开发者 DD 的个人主页，Vue 3 + Vite 构建。

## 技术栈

- Vue 3 (Composition API + `<script setup>`)
- Vite

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build   # 输出到 dist/
npm run preview # 预览构建结果
```

## 项目结构

```
src/
├── main.js
├── style.css
├── App.vue
└── components/
    ├── NavBar.vue
    ├── HeroSection.vue
    ├── ProjectCard.vue
    ├── ProjectsSection.vue
    ├── SkillsSection.vue
    ├── AboutSection.vue
    └── FooterSection.vue
public/
└── logos/          # 项目 logo 图片
```
